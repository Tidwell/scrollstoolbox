How are the prices generated?

Prices are thanks to Scrolls Trading Bulletin and ScrollsPC. If you choose to use ScrollsPC pricing, you will also be able to select "high," "low," or "median" (average of low and high) pricing.

These prices are updated about every hour, but will not be perfectly in-sync with the respective data sources for a short period of time after the source updates.

Why do you ask for my In-Game Name and Email on the account page?

In the future you will be able to reset your account password using your email should you forget it. Through the ScrollsGuide API will will also be implimenting various features that take advantage of knowing your In Game Name. Neither field will ever be required, and we will not retain any information if you delete it or solicit you via email.

Whats this site built with?

It uses Twitter's Bootstrap and AngularJS on the frontend (built via Yeoman), a small NodeJS server on the backend and MongoDB for the database. Real-time communication uses socket.io and the webserver uses express. The dark bootstrap theme is thanks to danneu.

The site is slow/buggy!

This is currently an Alpha version, there are expected to be bugs. Please Report any problems you may find. Performance will improve as development continues.
